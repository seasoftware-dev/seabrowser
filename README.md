# Sea Browser

Sea Browser is a privacy-focused web browser built with C++ and GTK3. It aims to provide a fast, secure, and distraction-free browsing experience.

## Features

*   **Privacy First**: Built-in tracking protection, HTTPS-only mode, and user agent switching.
*   **Modern Interface**: Clean, dark-themed UI designed for focus.
*   **Lightweight**: Powered by WebKitGTK for fast rendering and low resource usage.
*   **Native Integration**: Designed to integrate seamlessly with Linux and Windows.
*   **Sea Pages**: Fast, local internal pages for Settings, History, and New Tabs.

## Keyboard Shortcuts

*   **Ctrl+T**: Open New Tab
*   **Ctrl+W**: Close Current Tab
*   **Ctrl+R**: Reload Page
*   **Alt+Left**: Go Back
*   **Alt+Right**: Go Forward

## Privacy Features

Sea Browser includes valid privacy protections enabled by default:

*   **Intelligent Tracking Protection (ITP)**: Blocks cross-site trackers.
*   **Cookie Control**: Guidelines to block third-party cookies.
*   **User Agent Spoofing**: Option to use Firefox or Chrome user agents to reduce fingerprinting or bypass compatibility checks.
